## Visual style guidelines for infographics/carousels

### Recommended color system (Google Flow-aligned)

```css
/* Primary Colors */
--bg-dark-primary: #0A0E27;      /* Deep navy - main backgrounds */
--bg-dark-secondary: #1A1D2E;    /* Midnight blue - cards, sections */
--accent-primary: #00D9FF;       /* Electric cyan - CTAs, highlights */
--accent-secondary: #4DD0E1;     /* Bright cyan - hover, secondary elements */

/* Text Colors */
--text-primary: #FFFFFF;         /* Pure white - headlines, main text */
--text-secondary: #E0E0E0;       /* Light gray - body text */
--text-muted: #B0B0B0;          /* Medium gray - captions, metadata */

/* Gradient System */
--gradient-hero: linear-gradient(135deg, #0A0E27 0%, #1A1D2E 50%, #00CED1 100%);
--gradient-accent: linear-gradient(90deg, #00D9FF 0%, #6366F1 100%);
--gradient-subtle: linear-gradient(180deg, rgba(0,217,255,0.1) 0%, transparent 100%);
```

### Carousel design specifications

**Structure for 8-10 slide carousels:**
1. **Slide 1**: Hook + visual preview (thumbnail-worthy) - dark background with cyan accent
2. **Slides 2-3**: Problem/context setup - use gradient overlays
3. **Slides 4-7**: Value delivery (one prompt/tip per slide) - clean dark cards
4. **Slide 8-9**: Key insight or transformation result - before/after with cyan highlights
5. **Slide 10**: CTA + brand mark - gradient background

**Typography hierarchy:**
- Headlines: Bold sans-serif (Google Sans Display style), 28-36pt, #FFFFFF
- Subheadings: Semibold, 20-24pt, #00D9FF (cyan accent)
- Body text: Regular, 16-18pt, #E0E0E0 (light gray)
- Prompts/code: Monospace, 14-16pt, highlighted with subtle cyan glow
- Vietnamese readability: Line spacing 1.5-1.7 for clarity

**Visual elements:**
- Dark backgrounds (#0A0E27, #1A1D2E) with cyan accents
- Before/after split screens with cyan divider line
- Numbered steps with cyan circular badges
- Screenshot tutorials with cyan annotation overlays and arrows
- Cards with subtle cyan border glow effect
- Gradient overlays for depth (teal to purple)
- Floating elements with soft shadows

### Design patterns to embrace
- High contrast (dark backgrounds + bright cyan + white text)
- Cinematic gradient overlays
- Glowing neon-style accents on key elements
- Clean geometric shapes with subtle depth
- Smooth blur effects on background images
- Generous padding and white space
- Premium dark UI aesthetic

### Avoid these elements
- Warm earth tones (browns, oranges) - conflicts with cyan theme
- Overly bright or saturated backgrounds
- Cluttered layouts without breathing room
- Flat design without depth/shadows
- Pastel colors - not compatible with dark theme
